{% trans project_name=app_name %}Add to Homescreen{% endtrans %}

{% trans %}If you wish to have a shortcut to the dictionary on your smartphone homescreen, follow the instructions below.{% endtrans %}

{% trans %}The shortcut looks like an app and you can click right into the website without opening your browser.{% endtrans %}

  • {% trans %}Open the frontpage of the dictionary you want a shortcut to.{% endtrans %}
  • iPhone (Safari, Firefox, Chrome)
    1. {% trans %}Select the share icon in the browser.{% endtrans %}
    2. {% trans %}Scroll down and select "Add to Home Screen". You may need to drag the row from the right to get to the button.{% endtrans %}
  • Chrome
    1. {% trans %}Open the main menu in the browser{% endtrans %}
    2. {% trans %}A list of options appears. Select "Add to home page", a little down the list{% endtrans %}
    3. {% trans %}A new window appears asking you to add this website to your home screen. Click "Add".{% endtrans %}
  • {% trans %}Samsung Internet{% endtrans %}
    1. {% trans %}Open the main menu in the browser{% endtrans %}
    2. {% trans %}Click "Add page to" at the top right in the menu{% endtrans %}
    3. {% trans %}Choose "Home screen" in the window that appears.{% endtrans %}

{% trans %}The shortcut is now on your homescreen, and you can access the dictionary page directly by clicking on it.{% endtrans %}
